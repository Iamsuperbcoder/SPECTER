SPECTER is a React app with two parts: an AI layer and a verification engine. Claude turns a messy, inconsistent insurance directory into structured provider records and drafts the regulator complaint letter. The verification engine (plain JavaScript) then scores each provider from 0 to 100 using the official federal NPI check-digit algorithm, placeholder and out-of-state phone detection, shared-NPI and duplicate-listing detection, and address/specialty checks, weighted by the provider's clinical role. It rolls those up into a network ghost score, a county-level network-adequacy map, an insurer leaderboard, and an estimate of affected members.
